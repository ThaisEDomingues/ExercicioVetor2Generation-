Algoritmo Bubble Sort em Portugol (Ordem Decrescente)
Este repositório contém um exemplo prático de implementação do algoritmo de ordenação Bubble Sort (Ordenação por Bolha) utilizando a linguagem Portugol. O objetivo do programa é organizar um vetor de números inteiros de forma decrescente.

🚀 Sobre o Projeto
O código demonstra conceitos fundamentais de lógica de programação, como:

Manipulação de vetores (arrays).

Estruturas de repetição (para).

Estruturas condicionais (se).

Lógica de troca de valores (swap) com variável auxiliar.

🧠 Como funciona o algoritmo?
O Bubble Sort percorre o vetor várias vezes. A cada passagem, ele compara elementos adjacentes e os troca de lugar se estiverem na ordem errada.

Nesta implementação específica:

O programa inicia com um vetor desordenado de 10 números.

A condição se (numeros[j] < numeros[j + 1]) garante que os menores valores sejam "empurrados" para o final do vetor, resultando em uma lista decrescente (do maior para o menor).

🛠️ Tecnologias Utilizadas
Portugol: Linguagem voltada para o ensino de lógica de programação.

Portugol Studio: Ambiente de Desenvolvimento (IDE) recomendado para executar o código.

# Algoritmo de Processamento de Vetores (Soma, Média e Filtros)

Este repositório contém um programa desenvolvido em **Portugol** que realiza a leitura de um vetor de 10 números inteiros e executa diversas operações de análise de dados, como filtragem de índices, identificação de números pares e cálculos estatísticos básicos.

## 🚀 Funcionalidades

O algoritmo executa as seguintes etapas:
1.  **Entrada de Dados:** Solicita ao usuário 10 números inteiros.
2.  **Filtro de Índices:** Exibe apenas os elementos armazenados nas posições ímpares do vetor (índices 1, 3, 5, 7 e 9).
3.  **Filtro de Valores:** Identifica e exibe todos os números pares digitados pelo usuário.
4.  **Cálculo de Soma:** Soma todos os 10 valores inseridos.
5.  **Cálculo de Média:** Calcula a média aritmética simples dos valores (utilizando precisão real).

## 🧠 Lógica Aplicada

O código utiliza conceitos essenciais de programação:
*   **Vetores:** Armazenamento estruturado de dados.
*   **Laços de Repetição (`para`):** Utilizados para percorrer o vetor e acumular valores.
*   **Operador de Módulo (`%`):** Empregado para verificar a paridade dos números (`n % 2 == 0`).
*   **Tipagem de Dados:** Uso de `inteiro` para contagens e `real` para garantir a precisão decimal na média.

## 🛠️ Tecnologias Utilizadas

*   **Portugol**: Linguagem voltada ao ensino de lógica e algoritmos.
*   **Portugol Studio**: Ferramenta recomendada para rodar o código.

## 📝 Exemplo de Uso

**Entrada:** `2, 5, 1, 8, 4, 6, 7, 3, 10, 9`

**Saída esperada:**
*   **Elementos nos índices ímpares:** `5, 8, 6, 3, 9`
*   **Elementos pares:** `2, 8, 4, 6, 10`
*   **Soma:** `55`
*   **Média:** `5.5`
